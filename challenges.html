Bagaimana Saya beralih dari Youth Support Worker menjadi Web Designer dalam 8 Bulan

7 Juni 2023 | 6 Menit

Learning to code so that you can land a job in tech can feel daunting. That’s why we’re sharing inspiring stories from Codecademy’s community — to show how people like you (yes, you!) can embark on a learning journey and end up with a totally new career. We hope these stories serve as a reminder that there’s no single path to a more fulfilling work life.

Mengapa Aku memilih untuk belajar Coding

“Saya kuliah di perguruan tinggi lokal tempat saya mempelajari seni dan desain tradisional, dan saya menjadi pekerja pendukung pemuda di sebuah rumah kelompok selama satu setengah tahun. Saya bekerja bersama pekerja sosial, dan saya bertanggung jawab untuk mendukung klien. Saya harus mengajar banyak seni saya di sana, karena klien saya sangat artistik. Aku mengambil kursus desain UX — tetapi sebenarnya saya tidak menyelesaikannya. Ketika saya sampai di akhir proyek pertama saya, dan memulai proyek kedua saya, saya seperti, Oke, tapi saya ingin membangunnya. Saya ingin membangun apa yang saya buat. Saya tidak ingin berhenti hanya dengan desainnya. Saat itulah saya menemukan Codecademy dan saya seperti, Bagaimana saya bisa membangun situs web sekarang? Jadi, pada 1 Januari 2022, saya mendaftar ke Pro dan baru berkomitmen. Saya menyelesaikan kursus HTML hari itu, dan saya benar-benar mencetak sertifikat kelulusan* dan menggantungnya di dinding. Kemudian saya melakukan Learn CSS dan Learn JavaScript, dan saya mengikuti jalur karir Front-End Engineer.”

Bagaimana Saya meluangkan waktu untuk belajar

“Jadwal saya aneh karena saya bekerja dari jam 4 sore. sampai tengah malam. Pada dasarnya, ketika saya bangun di pagi hari, saat itulah saya akan belajar. Saya tidak terlalu disiplin, tapi saya juga seorang pelari, dan saya selalu mengikuti program latihan. Ketika saya memiliki program, saya menaatinya. Jadi itulah mengapa saya menyukai jalur karier Codecademy.”

Berapa lama waktu yang Saya butuhkan untuk mendapatkan perkerjaan

“Saya mulai melamar pekerjaan segera. Saya bepergian untuk bekerja di rumah grup setiap hari, tetapi saya ingin bekerja dari rumah dan menggunakan laptop saya. Ada semua pekerjaan jarak jauh di Kanada atau AS yang mendapatkan, seperti, ratusan pelamar. Saya tidak mendapatkan semua itu; Saya bahkan tidak mewawancarai sama sekali dalam beberapa bulan pertama karena saya tidak pernah sampai ke tahap itu. Kemudian saya bergabung dengan banyak grup seperti Facebook dan grup Codecademy di Discord. Saya sangat menikmati grup Discord, karena proyek dan hal-hal yang disarankan orang sangat spesifik.”

Bagaimana saya sampai

“Saya menjangkau bisnis lokal, karena saya tinggal di kota yang sangat kecil. Saya sudah selesai dengan HTML dan CSS, dan saya membangun melalui jalur Front-End Engineer, dan saya belajar JavaScript. Saya mengirim email ke begitu banyak orang di kota saya, tetapi hanya satu bisnis yang menghubungi saya dan setuju untuk menelepon. Saat menelepon, mereka berkata, 'Anda belum pernah bekerja dengan WordPress — itulah yang kami gunakan setiap hari.' Ini terjadi pada hari Senin. Mereka mulai menyelesaikan panggilan telepon, berkata, 'Saya tidak tahu. Hubungi kami kembali saat Anda mempelajari WordPress.’ Dan saya berkata, ‘Oke, saya akan memiliki sesuatu untuk Anda pada hari Jumat. Saya akan mengirimi Anda tautan. Minggu itu, saya terjun ke WordPress. Saya mengikuti semua tutorial, saya membeli domain dan melakukan hosting — dan saya belum pernah melakukannya sebelumnya. Saya mempelajari WordPress, membuat situs web satu halaman kecil ini, dan mengirimkannya kepada mereka. Saya pergi pada hari Jumat untuk menemui mereka dan mereka seperti, 'Kamu melakukan ini dalam 4 hari?' Dan saya seperti, 'Ya, saya belum pernah menggunakan WordPress sebelumnya. Apakah tidak apa-apa?’ Dan mereka seperti, ‘Ya, itu bagus.’”

Bagaimana saya melakukan wawancara

“Saya melakukan wawancara lain pada minggu berikutnya, tetapi itu lebih merupakan hal yang bersifat kepribadian. Mereka bertanya kepada saya, 'Anda benar-benar membuat situs web itu dalam 4 hari?' Saya seperti, 'Ya, saya melakukannya.'

Bagaimana saya mengevaluasi tawaran itu

“Sulit meninggalkan pekerjaan saya di rumah kelompok, karena saya harus mengajar seni saya kepada klien di sana - jelas itu bukan jenis seni yang saya lakukan di perguruan tinggi, tetapi setidaknya terkait. Dan itu sangat sulit karena saya harus bekerja dengan satu klien begitu lama dan mengenal mereka. Tetapi dengan pengkodean, saya seperti, Inilah yang benar-benar ingin saya lakukan selama bertahun-tahun, bukan hanya saat ini. Jadi itu sangat berharga.”

Bagaimana hari pertama dan seterusnya berlalu

“Beberapa hari pertama menyenangkan. Mereka memberi saya banyak pekerjaan kecil, seperti, Bisakah Anda mengubah tombol ini menjadi panjang yang berbeda? Kemudian saya akan kembali ke meja saya untuk mencari tahu sendiri, karena saya gugup ketika orang melihat dari balik bahu saya. Saya akan kembali dengan produk jadi, seperti, 'Apakah saya melakukan ini dengan benar?' Dan mereka akan berkata, 'Ya, Anda melakukannya dengan sempurna.' “Beberapa hari pertama menyenangkan. Mereka memberi saya banyak pekerjaan kecil, seperti, Bisakah Anda mengubah tombol ini menjadi panjang yang berbeda? Kemudian saya akan kembali ke meja saya untuk mencari tahu sendiri, karena saya gugup ketika orang melihat dari balik bahu saya. Saya akan kembali dengan produk jadi, seperti, 'Apakah saya melakukan ini dengan benar?' Dan mereka akan berkata, 'Ya, Anda melakukannya dengan sempurna.'

Apa yang saya harap saya ketahui sebelum saya mulai belajar

“Semua orang selalu mengatakan mereka berharap mereka memulai lebih awal. Tetapi Anda tidak pernah melakukannya, karena itu sangat menakutkan. Ada begitu banyak hal keren seperti itu untuk dipelajari, seperti, Apa itu domain? Bagaimana cara kerja internet? Sejujurnya, saya memberi tahu semua orang tentang pelajaran kecil Codecademy pertama yang Anda lakukan di tempat seperti, Apa itu internet?* Dalam pelajaran tersebut, mereka berbicara tentang bagaimana ada pipa di bawah tanah, dan bahwa hiu adalah masalah sebenarnya dengan saluran internet. Saya memberi tahu semua orang tentang ini, dan saya telah mengirimkannya ke begitu banyak anggota keluarga, seperti, 'Yang ini memberi tahu Anda saluran internet di bawah lautan!' Itu hal yang menyenangkan untuk diketahui. If you’re learning to code to get a job, start reaching out to people — because people love to talk about their journeys. Even in my small town, there’s so many groups that get together to have coding nights. Coding is such an individual thing, because you’re on your laptop alone. But it can also bring people together.”

All of our web development paths start with a broad overview of the internet’s infrastructure and how it works. Kailie is right: You don’t have to be an engineer to benefit from understanding how the internet works!
